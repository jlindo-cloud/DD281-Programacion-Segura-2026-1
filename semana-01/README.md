# Semana 01 — Introducción a la Programación Segura + Modelado de Amenazas

## Tema
Fundamentos de seguridad en software: Tríada CIA, OWASP Top 10, Principio de Mínimo Privilegio y Modelado de Amenazas con STRIDE.

## Competencia de la semana
Al finalizar, el estudiante será capaz de **identificar activos de software, clasificar amenazas usando STRIDE y aplicar la tríada CIA** para diseñar controles de seguridad básicos.
